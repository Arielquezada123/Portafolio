Centro de Control & Monitoreo

Visualización de métricas críticas mediante un dashboard desarrollado en Django, optimizado para el análisis de flujo ultrasónico. Se integran gráficos interactivos con actualización de latencia ultrabaja.

Resultado generado

Boleta digital generada automáticamente por el sistema a partir de las lecturas del medidor, lista para su distribución al cliente final.

Infraestructura de Borde (Edge)

Montaje físico del nodo sensor utilizando microcontroladores ESP32 y sensores de flujo. Este hardware actúa garantizando el procesamiento y depuración en el punto de captura del dato.

Despliegue de Sistema

Para asegurar la solidez y resiliencia en la telemetría, implementé un broker MQTT que canaliza el flujo desde el hardware hasta el servidor. Todo el entorno backend, la lógica principal y la persistencia de datos residen en AWS EC2, totalmente acoplados y transportables bajo contenedores Docker.
