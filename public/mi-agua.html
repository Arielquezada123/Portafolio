Proyecto: Mi Agua

Interfaz de Monitoreo en Tiempo Real

"Visualización de métricas críticas mediante un dashboard desarrollado en Django, optimizado para el análisis de flujo ultrasónico."

Infraestructura IoT y Sensores

"Montaje físico del nodo sensor utilizando microcontroladores ESP32 y sensores de flujo para captura de datos en el borde (Edge)."

Arquitectura del Sistema

Para garantizar la integridad en la búsqueda de soluciones óptimas, implementé un protocolo MQTT que asegura la telemetría desde el sensor hasta la instancia AWS EC2. Todo el entorno backend y las bases de datos operan bajo contenedores Docker para facilitar el despliegue y la escalabilidad.
